Part six Add the USWDS Identifier

Now we'll add the USWDS Identifier, just as we did the Banner.

Get the USWDS Identifier HTML.

Head over to the USWDS Identifier component page. Expand the Component code accordion and copy the first section of markup, from <div class="usa-identifier"...> to its closing </div>

Add the Identifier HTML into the tutorial markup.

Paste the Identifier markup into the bottom of the tutorial index file (index.html), right after the footer.

description index.html

...
</main>
{% include footer.html %}

<div class="usa-identifier">
  <section
    class="usa-identifier__section usa-identifier__section--masthead"
    aria-label="Agency identifier,"
    >

  ...

  </section>
</div>

Update image paths.

Just like we did in Part Five, we'll need to update the image paths. In index.html, search for all instances of the broken path (/assets/img) and replace it with the USWDS image path in our project (/assets/uswds/img).

This time, there's only one path to replace.

description index.html

src="/assets/img/circle-gray-20.svg"
src="/assets/uswds/img/circle-gray-20.svg"

And now you've got a working Identifier! It needs real data (which we won't add in this tutorial) but the formatting is all set.
