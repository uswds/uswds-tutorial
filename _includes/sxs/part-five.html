Part Five: Refine Finally, we’ll customize USWDS settings

In the last section, we'll bring it all home by customizing and optimizing our installation for better performance.

Activate theme settings.

We're going to add some custom USWDS settings to our project. To do this, we're going to add them to our custom settings file at _theme/_uswds-theme.scss by replacing the commented code with real settings.

There's a comprehensive list of settings on the USWDS website at designsystem.digital.gov/documentation/settings/, but for now, we’ll just modifiy our installation to turn off noisy notifications, use Public Sans as our sans-serif font, give our banner a dark gray background color, and make the banner expand the full width of the page.

description _theme/_uswds-theme.scss

//  @use "uswds-core" with (
//    $setting: value,
//    $setting: value
//  );
@use "uswds-core" with (
  $theme-show-notifications: false,
  $theme-font-type-sans: "public-sans",
  $theme-banner-background-color: "ink",
  $theme-banner-max-width: none
);

Let's compile with npx gulp compile to see these changes.

npx gulp compile

To finish up, we’ll optimize our installation.

Now we've added the USWDS Banner, but the size of our compiled CSS is quite large — over 500 KB! That's becase it contains every style for every component in the design system. We can greatly reduce the size by compiling only the Banner CSS.

To do this, we'll open up our theme's USWDS Sass entry point (_theme/styles.scss), and replace the main uswds package with the usa-banner package.

description _theme/styles.scss

@forward "uswds-theme";
@forward "uswds";
@forward "usa-banner";
@forward "uswds-theme-custom-styles";

Let's compile again to see the changes.

npx gulp compile

After we recompile, the CSS size drops down to 15 KB — less than 3% of the original size. It compiles in less than a second, and will gzip down to 2 KB in a production environment.

Now we have a project that uses USWDS code, includes the USWDS Banner, and is optimized to use only the USWDS code it needs. And that’s how you get started with USWDS code in a project!
