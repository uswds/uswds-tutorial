Compose the banner Let’s add the USWDS Banner

Get the USWDS Banner HTML.

Head over to the USWDS Banner component page. Expand the Component code accordion and copy the first section of markup, from <section class="usa-banner"...> to </section>

Add the Banner HTML into the tutorial markup.

We'll paste the Banner markup into the top of our default layout, just before the content.

description _layouts/default.html

---
---
      
<!doctype html>
<html lang="en">
{% include "site/head.html" %}
  <body class="{{ page.body }}">

    <section class="usa-banner" aria-label="Official website of the United States government">
      <div class="usa-accordion">
        <header class="usa-banner__header">

        ...

        </div>
      </div>
    </section>
    {{ content }}

Update image paths.

Now the banner is on our page. But when we look at it in the browser window, it doesn’t look right. The images are missing. That's because the image paths in our project are different from the image paths in the code we copied from the USWDS website — as many prjects will be. Let's update those image paths to our actual project paths.

In our banner code, we'll replace all instances of the broken path (/assets/img) with the our project's actual USWDS image path (/assets/uswds/img) using a find/change.

description _layouts/default.html

src="/assets/img/us_flag_small.png"
src="/assets/uswds/img/us_flag_small.png"


src="/assets/img/icon-dot-gov.svg"
src="/assets/uswds/img/icon-dot-gov.svg"


src="/assets/img/icon-https.svg"
src="/assets/uswds/img/icon-https.svg"

And once we've updated all these image paths, our banner should look perfect!
