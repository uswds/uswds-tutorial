Install First, let’s add USWDS and the USWDS compiler

Install the USWDS source code and compiler.

We’ll do this in a terminal window using a common package manager called npm.

npm install @uswds/uswds @uswds/compile --save-dev

Add a gulpfile to create your compile settings.

In this step, we’ll follow the USWDS compiler’s README to add what's called a "gulpfile" — instructions that tell the USWDS compiler how we’d like to use it.

touch gulpfile.js

In a text editor, open this new gulpfile we created, and import the USWDS compiler package.

This connects our gulpfile to the USWDS compiler and its functions.

description gulpfile.js

const uswds = require('@uswds/compile');

Set the proper USWDS version and path settings.

We're going to tell the compiler to use USWDS 3.x with a setting.

We'll also tell the compiler where to save USWDS source files, and the CSS we compile.

Today, we'll set the destination of our USWDS theme files to the ./_theme directory, and the destination of our other USWDS assets, like CSS, fonts, images, and JavaScript to an ./assets/uswds directory.

description gulpfile.js

const uswds = require('@uswds/compile');
      

uswds.settings.version = 3;

uswds.paths.dist.theme = './_theme';

uswds.paths.dist.css = './assets/uswds/css';
uswds.paths.dist.img = './assets/uswds/img';
uswds.paths.dist.fonts = './assets/uswds/fonts';
uswds.paths.dist.js = './assets/uswds/js';

Finally, define the USWDS compile functions we want to use.

The USWDS compiler has a number of functions available. To use these functions, we need to "export" them in a format we document in the compiler's README. We'll use these two newly defined functions — init, and compile — when we run the compiler in the next parts.

After adding these functions, we can save and close the gulpfile.

description gulpfile.js

const uswds = require('@uswds/compile');

uswds.settings.version = 3;

uswds.paths.dist.theme = './_theme';

uswds.paths.dist.css = './assets/uswds/css';
uswds.paths.dist.img = './assets/uswds/img';
uswds.paths.dist.fonts = './assets/uswds/fonts';
uswds.paths.dist.js = './assets/uswds/js';
  
exports.init = uswds.init;
exports.compile = uswds.compile;
