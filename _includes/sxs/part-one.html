Part One: Install Let’s add USWDS and the USWDS compiler

Open a new terminal window.

Open a new terminal window and navigate to our project’s root directory.

cd code/uswds/uswds-tutorial

Install the USWDS source code package.

npm install @uswds/uswds --save-dev

Install the USWDS compiler package.

npm install @uswds/compile --save-dev

Add a gulpfile to create your compile settings.

Now we'll follow the USWDS compiler's instructions. The compiler uses a application called Gulp to manage compiling. We'll need what's called a "gulpfile" to tell the USWDS compiler how we'd like to use it.

touch gulpfile.js

In the new gulpfile we just created, import the USWDS compiler package.

This connects our gulpfile to the USWDS compiler and its functions.

description gulpfile.js

const uswds = require('@uswds/compile');

Set the proper USWDS version.

We're going to use the current version of USWDS, USWDS 3. We'll tell the compiler which version of USWDS to use with a setting.

description gulpfile.js

const uswds = require('@uswds/compile');
      

uswds.settings.version = 3;

Then, add custom path settings.

We use path settings to tell the compiler where we want to save USWDS source files, and the CSS we compile. Paths are relative to the project root, the place where we made our gulpfile.

In this tutorial we'll set the destination of our theme files to the ./_theme directory, and the destination of our other USWDS assts, like compiled CSS, plus fonts JS, and images to a uswds directory in ./assets.

description gulpfile.js

const uswds = require('@uswds/compile');

uswds.settings.version = 3;
  
uswds.paths.dist.css = './assets/uswds/css';
uswds.paths.dist.img = './assets/uswds/img';
uswds.paths.dist.fonts = './assets/uswds/fonts';
uswds.paths.dist.js = './assets/uswds/js';
uswds.paths.dist.theme = './_theme';

Finally, define the USWDS compile functions we want to use.

The USWDS compiler has a number of functions available for compiling styles and scaffolding projects. To use these functions, we need to "export" them in a specific format we show in the compiler's docs. We'll use these newly defined functions — init, and compile — when we run the compiler in the next parts.

description gulpfile.js

const uswds = require('@uswds/compile');

uswds.settings.version = 3;

uswds.paths.dist.css = './assets/uswds/css';
uswds.paths.dist.img = './assets/uswds/img';
uswds.paths.dist.fonts = './assets/uswds/fonts';
uswds.paths.dist.js = './assets/uswds/js';
uswds.paths.dist.theme = './_theme';
  
exports.init = uswds.init;
exports.compile = uswds.compile;
