Compose Now, we’ll add the USWDS Elements Banner code to our project

Copy the JavaScript.

Here we'll move the newly installed USWDS Elements JavaScript from the installation location to our project directory.

We’ll do this by creating a directory for this USWDS code in our project, using the mkdir command in the terminal. But there are many ways to make a directory!

mkdir assets/uswds/js

Then we’ll copy the files, for simplicity, using the terminal as before, but you can always just drag and drop.

cp -r node_modules/@uswds/elements/dist assets/uswds/js

We’ve just added the USWDS Elements JavaScript to our new project directory. Now we have to add a link to this code in our project template.

Add USWDS Elements JavaScript to the end of the page body.

We’ll add it before the closing body tag in our default page layout with a script element.

description ./_layouts/default.html

...

<body class="{{ page.body }}">
  {{ content }}
  <script src="/assets/uswds/js/dist/components/usa-banner.js" type="module"></script>
</body>

Finally, we’ll add USWDS Elements markup to the page body as well.

Instead of copying 93 lines of code from the USWDS website, we'll add one simple line of markup — usa-banner opening and closing tags (<usa-banner></usa-banner>) — to our default layout.

description ./_layouts/default.html

...

<body class="{{ page.body }}">
  <usa-banner></usa-banner>
  {{ content }}
  <script src="/assets/uswds/js/dist/components/usa-banner.js" type="module"></script>
</body>

When we save and scroll to the top of the page we'll see the new banner. And that's all it takes to install a simple USWDS Banner into any project with USWDS Elements!
