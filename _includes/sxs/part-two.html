Part Two: Compose Now, we’ll add the USWDS Elements Banner code to our project

Copy the JavaScript.

Here we'll move the newly install USWDS Elements JavaScript from the install location to our project directory.

First we'll create a directory for this USWDS code in our project

mkdir assets/uswds/js

Then we’ll copy the files.

cp node_modules/@uswds/elements/dist/components/usa-banner.js assets/uswds/js

Add USWDS Elements JavaScript to the end of the page body.

We’ll add the Banner JavaScript (usa-banner.js) to the end of the page body.

description ./_layouts/default.html

...

<body class="{{ page.body }}">
  {{ content }}
  <script src="/assets/uswds/js/dist/components/usa-banner.js" type="module"></script>
</body>

Finally, we’ll add USWDS Elements markup to the page body.

Instead of copying 93 lines of code from the USWDS website, we'll add one simple line of markup (<usa-banner></usa-banner>) to our default layout.

description ./_layouts/default.html

...

<body class="{{ page.body }}">
  <usa-banner></usa-banner>
  {{ content }}
  <script src="/assets/uswds/js/dist/components/usa-banner.js" type="module"></script>
</body>

When we load the page in a new window we'll see the new banner. And that's all it takes!
