Part Two: Connect Now, we’ll set up USWDS theme files and assets

Initialize our USWDS installation.

Now we'll use the functions we set up in the last section to initialize USWDS. This process adds USWDS assets and theme files to our project, then compiles a USWDS stylesheet.

USWDS static assets — fonts, images, and JavaScript — support USWDS styles. Initialization copies USWDS assets out of the package we installed in the first part and into the project directories we just defined.

We'll use the init function to copy these assets and compile USWDS CSS.

npx gulp init

After the script runs, we have new USWDS assets in an ./assets/uswds directory, theme files in a ./_theme directory, and compiled CSS in the ./assets/uswds/css directory.
