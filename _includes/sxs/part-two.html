Initialize our USWDS installation.

Here we'll use the settings and functions we set up in the last section to initialize USWDS. This process adds USWDS assets and theme files to our project, then compiles a USWDS stylesheet.

We'll use the npx gulp init command in the terminal to copy these assets and compile USWDS CSS.

npx gulp init

After the script runs, we have new USWDS assets in an ./assets/uswds directory, theme files in a ./_theme directory, and compiled CSS in the ./assets/uswds/css directory.
